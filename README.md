# iFanzy Creator Recruitment

Creator recruitment landing page for iFanzy platform - showcasing AI-powered creator monetization.

## Version

**Current Version:** 1.0

## Files

- `creator_recruitment.html` - English version
- `creator_recruitment_tc.html` - Traditional Chinese version
- `creator_recruitment_en.pdf` - English PDF export
- `creator_recruitment_tc.pdf` - Traditional Chinese PDF export

## Features

- 14 responsive slides with smooth scroll snap
- Language switcher (EN/繁中) with position preservation
- AI-powered creator monetization showcase
- 8 revenue streams overview
- Creator control dashboard
- Safety and control features

## PDF Generation

To regenerate PDFs:

```bash
npm install
node generate_pdf.js
```

## Technology

- Pure HTML/CSS/JavaScript
- No external dependencies for the landing page
- Puppeteer + pdf-lib for PDF generation
